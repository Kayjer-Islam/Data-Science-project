Project Title: Early Heart Attack Risk Prediction for Young Adults

Overview
This project analyzes a real-world heart disease dataset to identify key risk factors for early heart attack prediction among young adults. It focuses on data cleaning, preprocessing, and exploratory data analysis to understand patterns in patient health data.

Technologies Used
R
tidyverse
Data Cleaning & Preprocessing
Exploratory Data Analysis (EDA)

Dataset
Combined dataset of 1,800+ patient records
Includes medical and demographic features such as:
Age, Sex
Blood Pressure (trestbps)
Cholesterol (chol)
Heart Rate (thalach)
Target variable indicates heart attack risk:
1 → High Risk
0 → Low Risk

Key Features
Introduced real-world data issues (missing values, outliers, duplicates)
Performed data cleaning using:
Median (numerical data)
Mode (categorical data)
Standardized inconsistent categorical values
Removed duplicate records
Conducted statistical analysis to compare risk factors

Analysis Highlights
Compared age, cholesterol, and blood pressure across risk groups
Analyzed cholesterol differences based on gender
Studied heart rate variation across chest pain types

Outcome
Successfully cleaned and prepared the dataset for analysis
Identified patterns related to heart attack risk
Demonstrated practical data preprocessing and analysis workflow
